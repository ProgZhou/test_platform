# 测试用例

## 一、测试用例管理 (CASE)

### TC-CASE-001: 手动创建测试用例（AC-1）
- **前置条件**: 已存在目录"项目A > 小组B > 需求C"
- **测试步骤**:
  1. 在"需求C"目录下点击"新建用例"
  2. 填写用例名称: "登录功能-正常登录"
  3. 填写前置条件: "用户已注册"
  4. 填写操作步骤: "1. 打开登录页 2. 输入用户名密码 3. 点击登录"
  5. 填写预期结果: "跳转到首页"
  6. 点击"提交"
- **预期结果**: 用例创建成功，出现在"需求C"目录的用例列表中
- **优先级**: P0

### TC-CASE-002: Excel 批量导入用例-正常导入（AC-2）
